# Kumpulan Contoh Query NLU (Natural Language Understanding)

File ini berisi daftar kalimat atau *test queries* (skenario pengujian) yang sudah dimasukkan dan dilatih ke dalam model NLU Rasa kita. 
Kamu bisa menggunakan kalimat-kalimat ini untuk mengetes seberapa akurat Chatbot dalam menangkap *Intent* (maksud) dan *Entities* (seperti budget, fitur, tipe mobil).
